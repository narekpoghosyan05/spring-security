# Spring Security Demo

A simple demo application to showcase **Spring Security** features, including authentication and authorization in a Spring Boot project.

## Features

- User login and logout
- Role-based access control (e.g., ADMIN, USER)
- Password encryption using BCrypt
- Protected routes with access restrictions
- Basic form-based authentication

## Technologies Used

- Java 17+
- Spring Boot
- Spring Security
- Maven/Gradle
- H2 Database (for demo)
- Thymeleaf (for frontend templates)

## Getting Started

### Prerequisites

- Maven or Gradle
